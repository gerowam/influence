The Debye functions D_n(x) are defined by the following integral, For further information see Abramowitz & Stegun, Section 27.1. The Debye functions are declared in the header file ‘gsl_sf_debye.h’.

Function: double gsl_sf_debye_1 (double x)
Function: int gsl_sf_debye_1_e (double x, gsl_sf_result * result)

These routines compute the first-order Debye function D_1(x) = (1/x) \int_0^x dt (t/(e^t - 1)).
